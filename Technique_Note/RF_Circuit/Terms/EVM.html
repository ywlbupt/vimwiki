误差向量幅度（EVM，Error Vector Magnitude）

TD-SCDMA技术规范定义了两种调制精度测量方法。

3Gpp规定发射机的调制质量用误差向量幅度（EVM，error vector magnitude）和峰值码域误差（PCDE，peak code domain error）来衡量。其中对于TD-SCDMA技术规范使用编码EVM，最大值为12.5%，PCDE<-28dB(如果码片速率1.28Mcps和3.84Mcps都采用扩频因子16，则PCDE应为-28dB) 。

  1. 峰值码域误差（PCDE，peak code domain error），用于评定玛道扩频及保持正交的性能。峰值码域误差要考虑码域噪声；如果码域噪声太高，会降低接收机正确去除信道相关性的能力。码域噪声应保持在最低水平，以确保每个用户的服务质量。
  2. 误差矢量幅度（EVM，error vector magnitude）。通过分析星座图并取测量点与参考位置之间的唯一作为误差向量，可评估调制精度。参考位置有参考信号决定。首先将接收信号解调成符号，在对这些符号重新调制成理想的参考星座图。误差矢量信号的均方根值以总信号幅度百分比表示。

  1. 输入I/Q信号幅度的不平衡，正交调制器移相误差（I/Q相位的不平衡）
  2. 载波泄漏
  3. 通道幅度不平坦性（重构滤波器的通带纹波，基带脉冲成形滤波器），通道相位失真（PCB走线长度不平衡，滤波器的相位失真不平衡）
  4. 本振相位噪声
  5. 非线性失真
